Charm
=====

Charm is interpreted DSL and runtime for writing/managing
closed-form high-level architecture models.

### Dependencies

numpy (v1.12.1)

scipy (v0.18.1)

merp (v0.11)

sympy (v1.0.1)

lmfit (v0.9.9)

networkx (v2.1)

For SMT capabilities:

z3 (v4.6.0 with python binding)

### Example Usage

```
python interpreter/parser.py models/adder.charm
```
